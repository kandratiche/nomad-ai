# Fonts

This project uses **Inter** and **Montserrat** via `@expo-google-fonts/inter` and `@expo-google-fonts/montserrat` (loaded in `app/_layout.tsx`). No manual download is required.

If you prefer to use local font files instead, place the following in this folder and update `_layout.tsx` to use `require()`:

- Inter: Light, Regular, Medium, SemiBold, Bold from [Google Fonts – Inter](https://fonts.google.com/specimen/Inter)
- Montserrat: Light, Regular, Medium, SemiBold, Bold, ExtraBold from [Google Fonts – Montserrat](https://fonts.google.com/specimen/Montserrat)
